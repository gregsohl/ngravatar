About
*****
NGravatar.Core provides methods for interacting with the Gravatar API on the .NET platform.
The project is hosted at http://code.google.com/p/ngravatar/.
It can also be installed with NuGet: PM> Install-Package NGravatar.Core.
See http://nuget.org/packages/NGravatar.Core for more information.
